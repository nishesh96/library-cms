Library-CMS

HOW TO RUN - 
SERVER
1) Open /server folder 
2) Run 'npm install' to load dependencies
3) Run 'npm start' to run the server

CLIENT
1) Run 'npm install' to load depenencies.
2) Run 'npm run build' to create build.
3) Run 'serve -s build' to serve the app.
